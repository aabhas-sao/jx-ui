### Linux

```shell
curl -L https://github.com/jenkins-x/jx-project/releases/download/v{{.Version}}/jx-project-linux-amd64.tar.gz | tar xzv 
sudo mv jx-project /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x/jx-project/releases/download/v{{.Version}}/jx-project-darwin-amd64.tar.gz | tar xzv
sudo mv jx-project /usr/local/bin
```
